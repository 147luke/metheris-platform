# Public Scope

## Purpose

This file defines the public disclosure boundary of the Metheris repository.

---

## Publicly Allowed Content

This repository contains:

- high-level descriptions of Metheris
- public positioning language
- non-operational architectural overviews
- conceptual system explanations
- general design principles
- non-sensitive portfolio documentation

---

## Excluded Proprietary Content

This repository does not contain:

- production source code
- private repositories or internal branches
- real implementation pipelines
- source-specific ingestion logic
- API endpoints or access details
- credentials, keys or tokens
- database schemas
- geospatial models
- risk-scoring methods
- alerting thresholds
- anomaly detection logic
- predictive model details
- LLM prompts or agent instructions
- dashboard JSON/configurations
- deployment scripts
- infrastructure diagrams with operational detail
- business model documentation
- pricing logic
- sales strategy
- client information
- partner information
- internal notes

---

## Repository Status

This repository is a public professional overview.

It is not a technical specification, open-source framework, reusable template, implementation guide, operational manual or full disclosure of the Metheris platform.
