# Security and Disclosure Policy

## Public Repository Boundary

This repository contains only protected public-facing documentation for the Metheris platform concept.

It must not be used to publish production code, operational infrastructure, private integrations, sensitive datasets, proprietary workflows, AI prompts, internal methods or commercial strategy.

---

## Do Not Publish

Never commit or disclose:

- API keys
- access tokens
- passwords
- private endpoints
- environment files
- production configurations
- customer or partner data
- restricted datasets
- source-specific ingestion logic
- dashboard configurations
- alert thresholds
- scoring formulas
- model weights or model configurations
- prompt chains
- internal agent instructions
- deployment scripts
- infrastructure secrets
- commercial documents
- legal agreements

---

## Protected Material

The following materials are proprietary and must remain outside the public repository:

- Metheris implementation strategy
- data acquisition methods
- data processing workflows
- geospatial analysis logic
- AI-assisted interpretation logic
- observability rules
- risk classification methods
- operational procedures
- UX/dashboard specifications
- business model and market strategy
- client-specific adaptations

---

## No Implementation Disclosure

This repository is intentionally non-operational.

Requests for source code, private pipelines, internal architecture, prompts, datasets, dashboards, infrastructure or implementation guidance are outside the public scope of this repository and require written authorization.
